### Ashvale Online

Inspired by the charm of classic online RPGs.

A browser-based medieval MMORPG prototype built with HTML, CSS, and JavaScript.

### Current Features

* 🏰 Explore Oakwatch Keep

* 🌳 Chop trees for logs

* 🪓 Gain Woodcutting XP

* 🎒 28-slot inventory system

* 🚶 Click-to-move movement

* 💬 NPC interaction system
